# CAHEDV
Computer-Assisted Heuristic Evaluation of Data Visualization - Flask App

## SETUP

Creating virtual environment  
```
python3 -m pip install --user virtualenv
python3 -m venv .venv
.\.venv\Scripts\activate
```
Install Flask
```
pip3 install flask
```

This install the our evaluation package  
```
cd package
pip3 install .
```
Run the App
```
cd ..
python3 -m flask run
```
